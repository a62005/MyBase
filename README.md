# MyBase

此專案用來記錄自己所寫的架構與學到的新東東

example分支有範例CODE

架構：
1. Kotlin+MVVM
2. BaseFragment、BaseActivity
3. 使用RequestManager進行網路請求，輸入BaseRequest，回傳BaseResponse；reqeust須實作BaseRequest定義Service(Retrofit架構)
4. Recycler.Adapter須實作BaseBindingAdapter

技術：
1. Navigation
2. Hilt
3. Room
4. Retrofit
5. Gson
6. Glide
